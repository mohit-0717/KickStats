# ⚽ KickStats

> Football intelligence, surfaced fast.

KickStats is a full-stack football analytics platform that delivers real-time match insights, player intelligence, transfer tracking, and AI-powered analysis through a modern, high-performance UI.

---

## 🔥 Features

### 🏟️ Landing Dashboard
- Featured match spotlight
- Global venue pulse (stadium insights)
- Latest transfer tape
- Scouting shortcuts
- Animated stats overview

---

### ⚡ Match Center
- Live match tracking
- Event timeline (goals, assists, cards)
- Referee analysis (strictness profile)
- Team comparison (radar charts)
- Player impact breakdown
- Venue pressure insights
- AI-powered match prediction (Neural Forecast)

---

### 🔍 Portals (Deep Dive)
- Player intelligence
- Club insights
- Transfer market
- Medical & injury tracking

---

### 👤 Player Detail
- Career timeline
- Transfer history
- Injury records
- Match performance logs
- AI scouting twins

---

### 🏆 Team Detail
- Standings
- Financial overview
- Sponsors
- Recent match performance

---

### 📊 Match Detail
- Match summary
- Full event timeline
- Momentum analysis
- AI oracle prediction

---

### 🤖 Pulse AI Assistant
- Natural language queries
- SQL-backed insights
- Football intelligence summaries
- Context-aware responses

---

### 🎮 Fan Cave
- Role-based UI
- Admin controls
- Live mutations (goals, transfers, injuries)

---

## 🧠 AI Capabilities

- Match outcome prediction
- Expected goals (xG-style logic)
- Tactical classification:
  - Dominant
  - Balanced
  - Volatile
- Player scouting intelligence

---

## 🏗️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Framer Motion (animations)
- Recharts (data visualization)

### Backend
- Java (Spring Boot)
- REST APIs
- MySQL Database

### AI Layer
- Python microservice
- Statistical + heuristic models
- AI-assisted query responses

---

## 🗄️ Database

- Relational schema (MySQL)
- Entities include:
  - Players
  - Teams
  - Matches
  - Events
  - Transfers
  - Injuries
  - Standings

---
